Bienvenue sur NumBaMal Partie2

Ici on genere le fichier de distance à partir du fichier de traces d'exécutions.

pour ce faire on lance make run qui contient la commande suivante:
./calcule_distance_parallele fichier.txt

fichier.txt représente le fichier contenant la trace d'exécution obtenue auparavant.

En sortie on aura donc notre fichier de distance utilisable pour la clusterisation.
